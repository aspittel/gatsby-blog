I started a blog last year where I write a post each week about a new technology. I decided I wanted to put everything in one place.

It is a static PWA generator for React.

It uses service workers to pre-cache key resources so you can eliminate dependence on the network.

They are super fast.

Gatsby loads only the critical HTML, CSS, data, and JavaScript so your site loads as fast as possible.

It also has plugins so the site works offline and has a `mainfest.json` which allows you to save the application like an app on your phone.
